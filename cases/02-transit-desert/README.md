# 02. 公共交通空白地域の抽出

## 概要

道路ネットワーク上の徒歩距離で全国を判定し、公共交通空白地域に居住する人口を定量化する。

## 利用データ

- 国土数値情報 道路データ（N13）
- 国土数値情報 バス停留所（P11）
- 国土数値情報 駅別乗降客数（S12）
- 令和2年国勢調査 125mメッシュ人口（e-Stat）（参考）

詳細は [`../../data/`](../../data/) を参照。

## 手法

[道路ネットワーク変換](../../common/road-network/)で生成したネットワーク上の徒歩距離で、鉄道駅・バス停からの到達範囲を判定。範囲外のメッシュ人口を集計して空白地域人口を推計。

## 結果

日本人の**約7人に1人（1,772万人・14.1%）**が公共交通空白地域に居住し、**65歳以上では567万人（高齢者の6人に1人）**にのぼることを定量化した。

## デモ

- https://shiwaku.github.io/japan-transit-desert-analysis-125/

## 課題

国土数値情報 バス停留所（P11）にはデマンドバス等のデマンド交通のバス停が含まれていないため、これらを考慮できていない。今後の国土数値情報の整備に期待したい。
