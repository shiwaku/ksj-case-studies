# 04. 津波避難シミュレーション（静岡県焼津市）

## 概要

津波避難施設の整備段階を変化させたときの、浸水域における避難困難率の変化を推計し、施設整備効果を定量評価する。

## 利用データ

- 国土数値情報 道路データ（N13）
- 100mメッシュ夜間人口（令和2年国勢調査・東大CSIS加工）（参考）※避難困難率の算定
- 静岡県第4次地震被害想定 レベル2重合せ図（参考）※津波到達時間（浸水深30cm）
- 国土地理院 指定緊急避難場所・指定避難所データ（参考）※避難場所
- 焼津市 津波一時避難施設（参考）※指定・民間協力ビル

詳細は [`../../data/`](../../data/) を参照。

## 手法

[道路ネットワーク変換](../../common/road-network/)で生成したネットワーク上で、浸水域内の各地点から避難施設への徒歩到達可否を判定。避難施設の整備段階（A → B → C）ごとに避難困難率を算出。

## 結果

津波避難施設の整備段階を A → B → C と変化させると、浸水域の避難困難率が **99.3% → 76.6% → 50.8%** まで低減すると推計し、施設整備効果を定量評価した。
